¿QUÉ ES INTERNETWORKING?

Internetworking es "el concepto de interconectar diferentes tipos de redes para construir una gran red global".

de manera que cualquier par de hosts conectados puede intercambiar paquetes .

Para construir una red interna, se necesita lo siguiente:

---> un esquema estandarizado para dirigir paquetes a cualquier host en cualquier red participante.

---> un protocolo estandarizado que define el formato.

---> el manejo de los paquetes transmitidos.

El sistema resultante de redes interconectadas se llama una interred , o simplemente un internet.

Internetworking es una combinación de las palabras inter ("between") y networking, (no funciona en internet ni en red internacional).
